# Omni Fabric v2 вҖ” дә‘з«Ҝ Agent еӣўйҳҹеҚҸдҪңе№іеҸ°

## 1. ж„ҝжҷҜе®ҡдҪҚ

### 1.1 дёҖеҸҘиҜқжҸҸиҝ°

дә‘з«Ҝ agent еӣўйҳҹеҚҸдҪңе№іеҸ°пјҢи®© AI agent дҪңдёәеӣўйҳҹжҲҗе‘ҳеҸӮдёҺејҖеҸ‘е·ҘдҪңжөҒпјҢйҖҡиҝҮ DAG зј–жҺ’е’ҢиҙЁйҮҸеҸҚйҰҲеҫӘзҺҜдҝқиҜҒжү§иЎҢиҙЁйҮҸгҖӮ

### 1.2 ж ёеҝғй—®йўҳ

AI й©ұеҠЁзҡ„ејҖеҸ‘зјәд№Ҹз»“жһ„е’ҢеҸҜйҮҚеӨҚжҖ§гҖӮжҜҸж¬ЎйғҪжҳҜд»Һйӣ¶ејҖе§Ӣзҡ„еҜ№иҜқпјҢжІЎжңүжөҒзЁӢгҖҒжІЎжңүйӘҢиҜҒгҖҒжІЎжңүз§ҜзҙҜгҖӮOmni Fabric v2 иҰҒи§ЈеҶізҡ„жҳҜпјҡ**и®© AI й©ұеҠЁзҡ„ејҖеҸ‘д»Һ"дёҖж¬ЎжҖ§еҜ№иҜқ"еҸҳжҲҗ"еҸҜйҮҚеӨҚгҖҒеҸҜиҝӯд»ЈгҖҒеҸҜдҝЎиө–зҡ„жөҒзЁӢ"**гҖӮ

### 1.3 зӣ®ж Үз”ЁжҲ·

зҶҹз»ғдҪҝз”Ё AI иҫ…еҠ©ејҖеҸ‘зҡ„дёӘдәәејҖеҸ‘иҖ…е’Ңе°ҸеӣўйҳҹпјҲ3-10 дәәпјүпјҢе·Із»ҸдёҚж»Ўи¶ідәҺеҚ•ж¬ЎеҜ№иҜқејҸ AI зј–зЁӢпјҢжғіиҰҒжҠҠеӨҚжқӮзҡ„ејҖеҸ‘д»»еҠЎпјҲз«ҜеҲ°з«ҜеҠҹиғҪејҖеҸ‘гҖҒbug дҝ®еӨҚпјүз»“жһ„еҢ–ең°дәӨз»ҷ agent жү§иЎҢпјҢе№¶еҜ№иҫ“еҮәиҙЁйҮҸжңүиҰҒжұӮгҖӮ

## 3. ж ёеҝғжҰӮеҝөжЁЎеһӢ

### 3.1 AgentпјҲжҷәиғҪдҪ“пјү

Agent жҳҜе№іеҸ°дёӯзҡ„"еӣўйҳҹжҲҗе‘ҳ"пјҢжңүиә«д»Ҫе’ҢиғҪеҠӣгҖӮ

```typescript
interface Agent {
  id: string;
  name: string;              // "Planner", "Implementer", etc.
  description: string;       // иғҪеҠӣжҸҸиҝ°
  avatar?: string;           // еӨҙеғҸ
  tags: string[];            // ["planning", "coding", "review"]
  cliBindings: CLIBinding[]; // з»‘е®ҡзҡ„ CLI еҗҺз«Ҝ
  defaultConfig: AgentConfig;
  isBuiltin: boolean;
}

interface CLIBinding {
  cliType: CLIType;          // "claude-code" | "codex" | "copilot" | ...
  model?: string;            // й»ҳи®ӨжЁЎеһӢ
  maxTurns?: number;
  allowedTools?: string[];
}

type CLIType = "claude-code" | "codex" | "copilot" | "gemini" | "openclaw";
```

### 3.2 HarnessпјҲзј–жҺ’пјү

Harness жҳҜдёҖдёӘ DAGпјҢе®ҡд№үдәҶ agent д№Ӣй—ҙзҡ„еҚҸдҪңжөҒзЁӢгҖӮ

```typescript
interface HarnessDefinition {
  id: string;
  name: string;
  description?: string;
  nodes: HarnessNode[];
  edges: HarnessEdge[];
  isTemplate: boolean;
  tags: string[];
  createdBy: string;         // user id
  teamId?: string;           // еӣўйҳҹеҪ’еұһ
}

interface HarnessNode {
  id: string;
  type: "agent" | "condition" | "gate" | "merge";
  position: { x: number; y: number };
  config: AgentNodeConfig | ConditionNodeConfig | GateNodeConfig | MergeNodeConfig;
}

interface AgentNodeConfig {
  agentId: string;           // з»‘е®ҡзҡ„ agent
  cliType?: CLIType;         // иҰҶзӣ– agent й»ҳи®Ө CLI
  promptTemplate: string;    // жҸҗзӨәжЁЎжқҝ
  constraints: Constraint[];
  maxRetries: number;
  onFailAction: OnFailAction; // "retry" | "route" | "abort"
  routeToNodeId?: string;    // еӨұиҙҘж—¶и·Ҝз”ұеҲ°зҡ„иҜҠж–ӯиҠӮзӮ№
  contextFilter?: ContextFilter;
  slotBindings?: SlotBinding[];
}
```

### 3.4 Quality LoopпјҲиҙЁйҮҸй—ӯзҺҜпјү

иҙЁйҮҸй—ӯзҺҜжҳҜ Omni Fabric v2 зҡ„ж ёеҝғе·®ејӮеҢ–пјҢз”ұеӣӣдёӘжңәеҲ¶з»„жҲҗпјҡ

1. **зәҰжқҹжЈҖжҹҘ** вҖ” е®ҡд№үеҸҜйҮҸеҢ–зҡ„иҙЁйҮҸж ҮеҮҶпјҢиҮӘеҠЁйӘҢиҜҒ agent иҫ“еҮә
2. **Agent дә’е®Ў** вҖ” дёҖдёӘ agent е®ЎжҹҘеҸҰдёҖдёӘ agent зҡ„иҫ“еҮәпјҢеҪўжҲҗеҸҚйҰҲеҫӘзҺҜ
3. **Gate иҠӮзӮ№** вҖ” дәәзұ»еңЁе…ій”®иҠӮзӮ№д»Ӣе…Ҙе®ЎжҹҘпјҢзЎ®и®Өж–№еҗ‘жӯЈзЎ®
4. **иҮӘеҠЁйҮҚиҜ•/и·Ҝз”ұ** вҖ” зәҰжқҹеӨұиҙҘж—¶иҮӘеҠЁи§ҰеҸ‘йҮҚиҜ•жҲ–и·Ҝз”ұеҲ°иҜҠж–ӯ agent

## 8. жҠҖжңҜеҶізӯ–и®°еҪ•

### 8.1 дёәд»Җд№ҲйҖү Go иҖҢдёҚжҳҜ Node.js

- дёҺ Multica дҝқжҢҒдёҖиҮҙзҡ„жҠҖжңҜйҖүжӢ©пјҢйҷҚдҪҺзӨҫеҢәи®ӨзҹҘжҲҗжң¬
- Go зҡ„е№¶еҸ‘жЁЎеһӢпјҲgoroutineпјүеӨ©з„¶йҖӮеҗҲ agent жү§иЎҢзҡ„е№¶иЎҢи°ғеәҰ
- еҚ•дәҢиҝӣеҲ¶йғЁзҪІпјҢз”ЁжҲ·е®үиЈ… daemon жӣҙз®ҖеҚ•
- жҖ§иғҪжӣҙеҘҪпјҢdaemon дҪңдёәеёёй©»иҝӣзЁӢйңҖиҰҒдҪҺеҶ…еӯҳеҚ з”Ё

### 8.2 дёәд»Җд№Ҳдҝқз•ҷ DAG иҖҢдёҚжҳҜиҪ¬еҗ‘ Issue й©ұеҠЁ

- DAG зј–жҺ’жҳҜ Omni Fabric зҡ„ж ёеҝғе·®ејӮеҢ–пјҢMultica жІЎжңү
- DAG еӨ©з„¶ж”ҜжҢҒиҙЁйҮҸеҸҚйҰҲеҫӘзҺҜпјҲзәҰжқҹжЈҖжҹҘгҖҒйҮҚиҜ•и·Ҝз”ұгҖҒдә’е®Ўпјү
- Issue й©ұеҠЁжӣҙйҖӮеҗҲжқҫиҖҰеҗҲзҡ„д»»еҠЎеҲҶй…ҚпјҢDAG жӣҙйҖӮеҗҲзҙ§иҖҰеҗҲзҡ„е·ҘдҪңжөҒ
- жңӘжқҘеҸҜд»ҘйҖҡиҝҮ"д»Һ Issue иҮӘеҠЁз”ҹжҲҗ DAG"жқҘжЎҘжҺҘдёӨз§ҚжЁЎеһӢ

### 8.3 дёәд»Җд№Ҳ Cloud + Local Daemon

- Agent CLI еҝ…йЎ»иҝҗиЎҢеңЁз”ЁжҲ·жң¬ең°пјҲйңҖиҰҒи®ҝй—®жң¬ең°д»Јз Ғеә“гҖҒgitгҖҒз»Ҳз«Ҝпјү
- дә‘з«ҜжҸҗдҫӣеҚҸдҪңгҖҒеӯҳеӮЁгҖҒе®һж—¶зҠ¶жҖҒеҗҢжӯҘ
- дёҺ Multica зӣёеҗҢзҡ„жһ¶жһ„жЁЎејҸпјҢз”ЁжҲ·зҗҶи§ЈжҲҗжң¬дҪҺ
- Daemon еҸҜд»ҘзӢ¬з«ӢиҝҗиЎҢпјҢж–ӯзҪ‘ж—¶д№ҹиғҪжү§иЎҢ

### 8.4 дёәд»Җд№Ҳе…Ёж–°е®һзҺ°иҖҢдёҚжҳҜиҝҒз§»

- зҺ°жңү Tauri + TypeScript д»Јз Ғеә“зҡ„жҠҖжңҜж ҲдёҺзӣ®ж Үе·®ејӮеӨӘеӨ§
- еҗҺз«Ҝд»Һ TypeScript иҝҒз§»еҲ° Go еҮ д№ҺжҳҜйҮҚеҶҷ
- еүҚз«Ҝд»Һ Vite + React Router иҝҒз§»еҲ° Next.js App Router д№ҹжҳҜйҮҚеӨ§еҸҳеҢ–
- е…Ёж–°е®һзҺ°еҸҜд»ҘйҒҝе…ҚиҝҒз§»иҝҮзЁӢдёӯзҡ„е…је®№жҖ§иҙҹжӢ…
- зҺ°жңүд»Јз Ғеә“зҡ„и®ҫи®ЎпјҲзҠ¶жҖҒжңәгҖҒзәҰжқҹжЈҖжҹҘгҖҒдёҠдёӢж–Үз»§жүҝпјүдҪңдёәеҸӮиҖғпјҢйҖ»иҫ‘з§»жӨҚиҖҢйқһд»Јз ҒиҝҒз§»
